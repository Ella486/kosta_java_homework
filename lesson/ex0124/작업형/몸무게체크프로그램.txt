【 문제 】
몸무게 검색, 몸무게 변경, 비밀번호 변경, 비밀번호체크 하는 프로그램을 만든다.
class 이름 : EncapsulationExam
class내부에 몸무게와 비밀번호를 저장 할 전역변수 2개 선언한다.
(비밀번호는 String, 몸무게는 int형 이며 같은 클래스 내부에서만 접근가능하게 만든다.)
몸무게 검색, 몸무게변경, 비밀번호변경, 비밀번호체크 하는 각각의 메소드를 선언한다.

▶ 몸무게검색 Method
     modifier   : 같은 클래스내부에서만 접근가능
     returnType : 알아서 처리한다.
     메소드이름 : getXxx()로 시작한다.
     인      수 : 비밀번호
     하 는  일 : 인수로 들어온 비밀번호를 가지고 비밀번호체크하는 메소드를 호출한 후 비밀번호가 일치하면 몸무게를 출력하고 일치하지 않으면 “비밀번호불일치”를 출력한다.
▶ 몸무게변경 Method
     modifier   : 같은 클래스내부에서만 접근가능
     returnType : 알아서 처리한다.
     메소드이름 : setXxx()로 시작한다.
     인      수 : 변경몸무게, 비밀번호
     하  는  일 : 인수로 들어온 비밀번호를 가지고 비밀번호체크하는 메소드를 호출한 후 비밀번호가일치하면 인수로 들어온 몸무게로 전역변수 값을 변경한다.
▶ 비밀번호변경 Method
     modifier   : 같은 클래스내부에서만 접근가능
     returnType : 알아서 처리한다.
     메소드이름 : getXxx()로 시작한다.
     인      수 : 기존비밀번호, 변경비밀번호
     하  는  일 : 인수로 들어온 기존비밀번호를 가지고 비밀번호체크하는 메소드를 호출한 후 비밀번호가일치하면 변경비밀번호로 전역변수 값을 변경한다.
▶ 비밀번호체크 Method
     modifier   : 같은 클래스내부에서만 접근가능
     returnType : boolean
     메소드이름 : isXxx()로 시작 한다.
     인      수 : 비밀번호
     하  는  일 : 인수로 들어온 비밀번호를 전역변수비밀번호와 비교한 후 일치하면 true를 일치하지 않으면 false를 리턴 한다.

【 실행방법 】
     -Scanner를 이용한다.
      1. 몸무게 검색     2. 몸무게변경    3. 비밀번호 변경      4.종료
           메뉴가 1 인 경우 : 비밀번호입력받는다.
           메뉴가 2 인 경우 : 비밀번호와 변경 몸무게 입력받는다.
           메뉴가 3 인 경우 : 기존비밀번호와 변경비밀번호를 입력받는다.
           메뉴가 4 인 경우 : 프로그램 종료한다. - System.exit(1) ;
